# MetaLamp_practiceProject
Just a webCraft-practice-project

-- Software --
Webpack 5.0
Pug
SCSS

-- Structure info --
src/ - includes all templates and entry files.
production/ - includes completed page bundles

-- Terminal Actions --
To install all node_modules used for this bundle run npm install
To create development-bundle folder run $npm run dev
To create product-bundle folder run $npm run build
To start local-server run $npm run start
